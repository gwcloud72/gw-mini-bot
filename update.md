마지막 갱신: 2026-07-15

## 저장소 역할

이 저장소는 공개 가능한 React·Tailwind 프런트엔드와 GitHub Pages 배포 설정만 포함합니다.


## 누적 변경 이력

### 2026-07-11 · UI 정리

- React 컴포넌트 구조를 분리했습니다.
- 현대적인 집중형 채팅 레이아웃, 계절 스킨, 접근성, 애니메이션을 적용했습니다.
- 폰트 크기·줄높이·자간을 역할 기반 타이포그래피 토큰으로 통일했습니다.
- 브라우저 아이콘과 서비스 표시 이름을 `미니챗`으로 통일했습니다.
